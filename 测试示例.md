## 线性回归的损失函数

线性回归使用**最小二乘法**，其损失函数定义为：

$$
L(\theta) = \frac{1}{2m} \sum_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)})^2
$$

其中 $h_\theta(x) = \theta^T x$ 是假设函数。

### 梯度下降更新公式

对 $\theta_j$ 求偏导：

$$
\frac{\partial L(\theta)}{\partial \theta_j} = \frac{1}{m} \sum_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)}) x_j^{(i)}
$$

更新规则：

$$
\theta_j := \theta_j - \alpha \frac{\partial L(\theta)}{\partial \theta_j}
$$

## Python 代码实现

```python
import numpy as np

def linear_regression(X, y, lr=0.01, epochs=1000):
    m, n = X.shape
    theta = np.zeros(n)
    
    for epoch in range(epochs):
        predictions = X @ theta
        errors = predictions - y
        gradient = (X.T @ errors) / m
        theta -= lr * gradient
        
        if epoch % 100 == 0:
            loss = np.mean(errors ** 2) / 2
            print(f"Epoch {epoch}, Loss: {loss:.6f}")
    
    return theta
```

> 提示：学习率 $\alpha$ 的选择对收敛速度影响很大。

### 常用学习率取值

- 太大：可能不收敛
- 太小：收敛速度慢
- 推荐：$0.001 \leq \alpha \leq 0.1$

## 总结

线性回归是机器学习中最基础的模型之一，其核心思想是通过最小化 **MSE** 来找到最优参数。
